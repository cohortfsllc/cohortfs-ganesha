Paths and handles

FSAL layer of ganeshell makes it possible to address entries using their full or relative path. However, you can also address them using their FSAL handle (hexadecimal representation preceded with a '@' sign): 

ganeshell> cd /
Current directory is "/" (@A552000000000000CD03A747FFFFFFFF020800000000000002000000000000001B000000300000007400A747000000000400000000000000)

# is equivalent to:
ganeshell> cd @A552000000000000CD03A747FFFFFFFF020800000000000002000000000000001B000000300000007400A747000000000400000000000000

The 'ls' command has a special option '-S' for displaying objects handle.
