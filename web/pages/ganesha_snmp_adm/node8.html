Using the SNMP client provided with GANESHA

Even if GANESHA statistics can be browsed using standard SNMP commands (snmp_get, snmp_walk...), GANESHA comes with a SNMP client tool (snmp_adm) for easily browsing those stats in a convivial way, without having to handle OIDs, and all SNMP relative stuff.

It is located in the 'snmp_adm/client' directory of GANESHA's distribution, and is also available in GANESHA RPMs that include SNMP support.
