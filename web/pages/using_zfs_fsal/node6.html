Destroying a zpool

To destroy a zpool only one command is needed: 

  root@localhost% lzw_zpool destroy pool
  root@localhost% lzw_zpool list
      NAME   SIZE  ALLOC   FREE    CAP  DEDUP  HEALTH  ALTROOT
      tank  4.59G  2.02G  2.57G    43%  1.00x  ONLINE  -
